# 🏦 Bank Management System (Banka Yönetim Sistemi)

[English] | [Türkçe](#türkçe)

---

## 🇺🇸 English

This is a file-based banking simulation developed in C for the **YZM103 Introduction to Algorithms and Programming** course.

### ✨ Features
*   **Secure Masked Password:** Passwords are hidden with `*` during input for security.
*   **File-Based Data Management:** User accounts, balances, and IBANs are persistently stored in `kullanicilar1.txt`.
*   **Automatic IBAN Generation:** Assigns unique, random IBANs to new users.
*   **EFT/Wire Transfer:** Supports peer-to-peer money transfers via IBAN.

### 🛠 Tech Stack
*   **Language:** C
*   **Environment:** Windows Console (UTF-8)

### 🚀 Quick Start
1. Compile the source: `gcc main.c -o bank_app`
2. Run the application: `./bank_app`
